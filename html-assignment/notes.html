HTML Tags description

P
This tag represents a paragraph.Paragraphs are usually rendered as blocks of text separated by blank lines. It can be any group of structured information.
Section
This tag it is used to create independent sections on a web page.
Nav
The purpose of this tag is to generate navigation links, either within the document or to other documents.
Strong
This tag represents high importance in the information or urgency.
Blockquote
Represents a large stand alone quote. Visually represented as indentation
